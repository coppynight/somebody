# Voice Twin Prototype

移动端 H5 语音数字人原型。

## 项目内容
- 一次性创建流程：创建助理通话采集 -> 建模仪式 -> 通讯录首页
- 日常使用流程：通讯录发起语音电话 -> 电话式通话界面
- 扩展入口：添加数字人、设置（查询/更新/分享）

## 文件结构
- `prototype.html`：主原型开发文件
- `pageone/index.html`：部署到 CloudBase Hosting 的页面入口
- `PRODUCT_PLAN.md`：产品方案
- `docs/PRD.md`：产品需求文档
